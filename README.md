# Core Cutting Construction Website

This is a beginner-friendly frontend website made with HTML and CSS for my father's construction business (Core Cutting & RCC Work), designed and built by me.

## 🔧 Features
- Clean modern layout
- Responsive design (basic)
- About, Services, and Contact section

## 📁 Project Structure
- `index.html` — Home page
- `style.css` — Stylesheet
- `/images` — Business-related images
